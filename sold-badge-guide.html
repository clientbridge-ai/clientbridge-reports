Add the SOLD Badge Code

functions.php · or use a snippets plugin

Where to add this

Go to your WordPress admin and navigate to:

› Appearance → Theme File Editor → functions.php

Or use a plugin like WPCode to add snippets safely without editing theme files.

PHP · functions.php 
add_action('woocommerce_before_shop_loop_item_title', 'add_sold_badge_loop', 5);
add_action('woocommerce_before_single_product_summary', 'add_sold_badge_single', 5);

function add_sold_badge_loop() {
    global $product;
    if (!$product->is_in_stock()) {
        echo '<span class="sold-badge">SOLD</span>';
    }
}

function add_sold_badge_single() {
    global $product;
    if (!$product->is_in_stock()) {
        echo '<span class="sold-badge-single">SOLD</span>';
    }
}

Add Sold Lead Capture

Turn sold product pages into inquiry generators

The concept

When a unit sells, the page keeps ranking in Google. Instead of showing a dead end, show a lead capture section that converts that traffic into buyer inquiries for similar equipment.

PHP · functions.php — Lead Capture Section 
add_action('woocommerce_single_product_summary', 'sold_lead_capture', 35);

function sold_lead_capture() {
    global $product;

    if (!$product->is_in_stock()) {
        echo '<div class="sold-lead-box">
            <h2>This Unit Has Been Sold</h2>
            <p>We frequently source similar equipment. Submit
            the form below and our team will help you locate one.</p>
            <a class="sold-button" href="/contact">
                Request Similar Equipment
            </a>
        </div>';
    }
}
CSS · Lead Capture Box Styling 
.sold-lead-box {
    border: 3px solid #c00000;
    padding: 30px;
    margin-top: 30px;
    background: #fafafa;
    text-align: center;
}

.sold-lead-box h2 {
    color: #c00000;
    font-size: 28px;
    margin-bottom: 10px;
}

.sold-button {
    background: #003087;
    color: #fff;
    padding: 14px 28px;
    display: inline-block;
    margin-top: 10px;
    text-decoration: none;
    font-weight: bold;
}

SEO Strategy for Sold Pages

Keep pages indexed · Build topical authority · Capture long-tail traffic

⚠ IMPORTANT: Do NOT delete sold product pages. They continue ranking for specific model searches long after the unit sells.
  • ✓
    Keep pages indexed. Sold product pages rank for long-tail searches like "Caterpillar 500kW generator for sale" and "used 3512 generator price" — searches that signal real buyer intent.
  • ✓
    Add internal links to sold pages. Create a "Recently Sold Equipment" section and link between related machines. This builds topical authority in your category.
  • ✓
    Remove the category archive from your sitemap. Instead of removing sold product pages, remove the /sold-products/ category listing from your XML sitemap so Google indexes the individual pages but not the archive listing.
  • ✓
    Build category landing pages. Create SEO pages for high-value searches and link sold machines to them as supporting pages.

Category Page Examples

Create dedicated pages for high-value buyer searches and link from sold product pages:

/generators/500kw-generator-for-sale/
/generators/caterpillar-generators/
/generators/diesel-generators-for-sale/

Optional Improvements

High-ROI upgrades for equipment dealer sites

📂

Auto-categorize sold items

Automatically move sold products into a "Sold Equipment" category to organize your inventory and build SEO archives.

🗺️

Remove from product sitemap

Remove the sold category archive from your Google product sitemap — but keep individual product pages indexed.

🔗

View Similar Equipment button

Add a button on each sold page linking to in-stock equipment in the same category, power range, or brand.

📄

Recently Sold pages for SEO

Create archive pages like "Recently Sold Caterpillar Generators" — these attract serious buyers researching the market.

Expected results

Equipment dealers using this system typically see 30–60% more organic leads, long-tail model search traffic, and inbound inquiries from buyers requesting similar units.

TIP: The highest-ROI page on your site may be a sold product that still ranks for an exact model number a buyer is searching today.
